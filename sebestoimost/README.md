# SebestoimostApp (GUI) — расчёт себестоимости контейнера

## Что в папке

- `SebestoimostApp_main.py` — основное приложение (Tkinter GUI)
- `cbm_map.json` — справочник “размер → CBM на 200 шт”
- `requirements.txt` — зависимости
- `RUN_APP_WINDOWS.bat` — запуск из исходников (Windows)
- `BUILD_EXE_WINDOWS.bat` — сборка `.exe` (Windows)

## Быстрый запуск (Windows)

Двойной клик по `RUN_APP_WINDOWS.bat`

или вручную:

```bat
py -m pip install -r requirements.txt
py SebestoimostApp_main.py
```

## Сборка EXE (Windows)

Двойной клик по `BUILD_EXE_WINDOWS.bat`.

Готовый файл появится в:

```
dist\SebestoimostApp.exe
```

⚠️ Важно: если вы меняете `SebestoimostApp_main.py`, то старый `.exe` НЕ обновится сам.
Нужно заново запускать сборку.

## Drag&Drop (опционально)

Drag&Drop включится автоматически, если установить библиотеку:

```bat
py -m pip install tkinterdnd2
```

Если библиотека не установлена — приложение просто работает без DnD.

## Как редактировать CBM

Откройте `cbm_map.json` (это обычный JSON-словарь):

```json
{
  "MS": 0.0186,
  "M": 0.026208
}
```

Ключ — это “размер/маркер” в наименовании.
Поддерживаются multi-word ключи (например `"B 2XL"`).

Алгоритм матчится по словам (последовательность токенов), с приоритетом самых длинных ключей.

## Как берутся параметры (лист 2)

Приложение ищет на листе параметров подписи (Фрахт / Терминалка / Курс ГТД и т.д.)
и рядом с ними числовые значения. Формат “в колонке A подпись, в B значение” тоже поддержан.

Если параметр не найден, ошибка покажет “первые найденные подписи” — по ним легко добавить синоним.
